# Address Book

## Overview
The Address Book is a command-line application developed in C that enables users to efficiently store, manage, and organize contact information. The project demonstrates core programming concepts such as structured data handling, modular design, and file operations.

## Features
- Add new contacts with relevant details  
- View all stored contacts  
- Search contacts by name or phone number  
- Edit existing contact information  
- Delete contacts  
- Persistent data storage using file handling  

## Technologies Used
- Programming Language: C  
- Concepts: Structures, File Handling, Modular Programming (Functions), String Manipulation  

## Project Structure
Address-Book/  
│── main.c  
│── contact.c  
│── contact.h  
│── file.c  
│── file.h  
│── contacts.txt  

## How to Run
Compile the program:
gcc main.c contact.c file.c -o address_book  

Run the program:
./address_book  

## Sample Functionality
The program provides a menu-driven interface allowing users to perform operations such as adding, viewing, searching, updating, and deleting contact records.
